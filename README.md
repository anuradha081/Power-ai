# Power-ai

# AI-Based Power Loss Prediction & Theft Detection

This project uses Machine Learning to:

• Predict power transmission losses  
• Detect abnormal losses indicating theft or faults  

## Technologies Used
- Python
- Pandas
- NumPy
- Random Forest
- Isolation Forest

## How it works
1. Predict expected power loss from system conditions
2. Compare with actual measured loss
3. Detect abnormal gaps using anomaly detection

## Run the project
Install dependencies:
pip install -r requirements.txt

Run:
python model.py
